# QuantLibXlOil

This is the documentation for the QuantLibXlOil Python package.

Source code is available at [GitHub](https://github.com/frame-consulting/QuantLibXlOil).

QuantLibXlOil is an interface package to make functions of the open-source [QuantLib](https://github.com/lballabio/QuantLib) pricing library available in Excel.

QuantLibXlOil is made possible by the [xlOil framework](https://github.com/cunnane/xloil).
